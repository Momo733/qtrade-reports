## 运行配置

> 配置来源：报告生成时的 bin/.env（trader 运行时 env，非当日快照）

### 模式与窗口

| 参数 | 值 | 含义 |
|---|---|---|
| TRADING_MODE | live | 模式：paper / live |
| SYMBOL | QQQ.US | 交易标的 |
| START_TIME | 04:00 | 允许开仓窗口起 (ET) |
| END_TIME | 15:50 | 强制平仓时间 (ET) |
| ENTRY_CUTOFF_MINUTES | 60 | 收盘前停止开仓的分钟数 |
| CHECK_INTERVAL | 5 | 主循环采样间隔（秒） |
| MAX_BAR_AGE_SECONDS | 120 | K 线最大允许滞后（秒） |
| ALLOW_EXTENDED_HOURS | true | 是否允许盘前盘后行情参与判定 |
| OPTION_EXTENDED_HOURS | false | 是否允许盘前盘后期权下单 |

### 仓位 & 风控

| 参数 | 值 | 含义 |
|---|---|---|
| CAPITAL | 100000 | 名义资金 (USD) |
| POSITION_PCT | 1 | 单笔预算占名义资金的百分比 |
| MIN_CONTRACTS | 1 | 单笔最少合约数 |
| MAX_TRADES | 50 | 当日最多开仓笔数 |
| DAILY_LIMIT_ENABLED | false | 当日亏损熔断开关 |

| ENTRY_COOLDOWN_SECONDS | 60 | 同标的撤单后冷却期（秒） |
