# Personal time-series notes

A personal logbook for daily data observations and visualizations. Auto-generated
content; manual edits to this repository will be overwritten on the next sync.

Live site: <https://momo733.github.io/qtrade-reports/>

## Layout

```
daily-trades/
  2026-MM-DD.md           # daily note (markdown, Jekyll-rendered to HTML)
  2026-MM-DD-chart.html   # interactive 1-min chart with markers
assets/
  plotly-2.35.2.min.js    # shared chart library (referenced by every daily html)
index.md                  # auto-regenerated by the publisher; lists all dates
```

## How updates happen

A cron job on a private server regenerates and pushes content here every weekday
after market close. The publisher is one-way: this repository is a derived view
of a private source of truth. Do not edit files here by hand — they will be
overwritten on the next push.
